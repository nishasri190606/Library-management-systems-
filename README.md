Library Management System with Book Issue Tracking
Project Title
Library Management System using C++
Objective
The main objective of this project is to develop a simple library management system that helps a librarian manage books, register members, issue books, return books, and calculate fines for late returns.
Features
Add new books to the library catalog
Register new members
Issue books to members
Return books and calculate fine
View available books
Store records using file handling
Technologies Used
Programming Language: C++
Concepts Used:
Classes and Objects
File Handling
Functions
Conditional Statements
Loops
Classes Used

Used to store book details:
Book ID
Title

Used to perform operations:
Issue Book

Files Created
books.txt → Stores book details
members.txt → Stores member details
issue.txt → Stores issued book records
How to Run
Open the program in any C++ compiler.
Compile the code.
Run the program.
Select options from the menu.
Sample Menu

Register Member
Issue Book
Return Book

Fine Calculation
Fine = Rs.5 per late day
Example:
If a book is returned 3 days late:
Fine = 3 × 5 = Rs.15
